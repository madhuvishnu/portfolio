# Portfolio

Production-ready static portfolio website built with plain HTML and CSS.

## Run locally

Open `index.html` directly in your browser.

## Publish on GitHub Pages

1. Push this project to a GitHub repository.
2. In GitHub, open `Settings` -> `Pages`.
3. Under `Build and deployment`, choose:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` (or your default branch)
   - `Folder`: `/ (root)`
4. Click `Save`.
5. Wait for deployment, then open the provided `https://<username>.github.io/<repo>/` URL.

## Customize

- Update contact values in `index.html`.
- Tune colors and spacing in `style.css` variables.
